Buatkan admin dashboard Nuxt 3 + Tailwind CSS untuk sistem Dialog Kebangsaan.

Dashboard harus memiliki:
- Card total pendaftar
- Card total hadir
- Card total survey
- Card total action plan
- Card total eligible sertifikat
- Chart/list jumlah peserta per sekolah
- Filter hari acara: semua, 17 Juni, 18 Juni
- Tabel peserta dengan search dan filter status
- Tombol export CSV/Excel
- Tampilan mobile/tablet/desktop responsif

Data berasal dari Supabase PostgreSQL.

Gunakan design clean, simple, warna merah putih navy.
Jangan pakai library UI berat jika tidak perlu.

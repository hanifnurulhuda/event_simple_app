Ingat, ini project MVP budget 2 juta. Jangan membuat fitur enterprise.

Batasi sistem pada:
- Registrasi peserta
- Check-in sederhana
- Survey 14 pertanyaan
- Sertifikat HTML printable
- Action Plan submission menggunakan link Google Drive
- Admin dashboard
- Export CSV/Excel
- Status tracking

Tidak perlu:
- WhatsApp gateway otomatis
- Upload video/foto langsung
- Generate PDF massal
- Multi-role kompleks
- Realtime websocket
- VPS deployment
- Queue worker
- Payment gateway
- Sistem scoring juri kompleks

Buat sistem yang cepat selesai, mudah dipakai, dan tidak berat saat event.
